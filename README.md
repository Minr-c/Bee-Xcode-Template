Bee-Xcode-Template
==================

Xcode Template for BeeFramework

## install

1. download this [package](https://github.com/qfish/Bee-Xcode-Template/archive/master.zip) or clone this repo.
2. enter the dir open dir with terminal.
3. 
  ```sh
    sh install.sh
  ```

## uninstall

```sh
  sh uninstall.sh
```
